# ecommerce-sql-powerbi-analysis
Title: E-commerce data analysis project using SQL and Power BI

Tools: Sql Server
       Power Bi

Dataset:  https://www.kaggle.com/datasets/sonalisingh1411/e-commerce-and-logistics-dataset

Process: Created relational database (customers, orders, products, etc.)
         Performed data cleaning and transformation
         Wrote SQL queries for analysis
         Built interactive Power BI dashboard
         
Key Insights: São Paulo (SP) contributes the highest number of orders and revenue
              Certain product categories dominate sales (e.g., home & furniture)
              Delivery delays vary significantly across states
