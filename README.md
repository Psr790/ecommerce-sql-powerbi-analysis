# E-commerce Data Analysis using SQL & Power BI

## 📌 Project Overview
This project analyzes an e-commerce dataset to understand customer behavior, revenue trends, and delivery performance.

## 🛠 Tools Used
- SQL Server (SSMS)
- Power BI

## 📊 Dataset
[Kaggle Dataset](https://www.kaggle.com/datasets/sonalisingh1411/e-commerce-and-logistics-dataset)

## ⚙️ Process
- Created relational database (customers, orders, products, etc.)
- Performed data cleaning and transformation
- Wrote SQL queries for analysis
- Built interactive Power BI dashboard

## 🔍 Key Insights
- São Paulo (SP) contributes the highest number of orders and revenue  
- Certain product categories dominate sales  
- Delivery delays vary significantly across states
